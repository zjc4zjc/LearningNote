# python的常用指令教程

本文记录 `python` 的常用指令。

前置条件：
- `python` 或者 `conda` 已经安装可用
在终端按行输入：

```bash
conda --version
(window请输入)where conda
(linux请输入)which conda
```

如果有类似的输出：

```text
conda 25.5.1

D:\Software\anaconda3\condabin\conda.bat
D:\Software\anaconda3\Scripts\conda.exe
```

说明conda能正常调用，可以进入某个虚拟环境唤起python。否则请查看教程[tut_conda的下载安装](tut_conda的下载安装.md)

## Tensor专栏

| 操作                | 示例代码                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 掩码                | `x = torch.tensor([1, 2, 3, 4, 5])`  <br>`mask = x > 2  # 生成一个布尔掩码`<br>`print(mask)   # tensor([False, False,  True,  True,  True])`<br><br>`# 用布尔掩码选出大于 2 的值filtered_x = x[mask]`  <br>`print(filtered_x)  # tensor([3, 4, 5])`<br><br>`# 用布尔掩码选出大于 2 的值,并赋值为0`  <br>`x[mask]=0`<br>`print(x) # tensor([1, 2, 0, 0, 0])`                                                                                                                                                                                                                                  |
| 生成                | `p=torch.tensor([[1,3,3],[5,0,2],[0,5,4]],dtype=torch.float32)`<br><br>在创建tensor时，你还可以指定tensor的设备。如果你不指定，默认是在CPU/内存上。如果你想创建一个GPU/显存上的tensor。可以通过把device关键字设定为“cuda”来指定。<br><br>`t_gpu = torch.tensor([1,2,3],device="cuda")`<br><br>`shape = (2,3)`  <br>`rand_tensor = torch.rand(shape) # 生成一个从[0,1]均匀抽样的tensor`<br>`randn_tensor = torch.randn(shape) # 生成一个从标准正态分布抽样的tensor`<br>`ones_tensor = torch.ones(shape) #生成一个值全为1的tensor`<br>`zeros_tensor = torch.zeros(shape) # 生成一个值全为0的tensor`<br>`twos_tensor = torch.full(shape, 2) #  生成一个值全为2的tensor` |
| 交换维度              | `x_transform = x.permute(0,1,3,2) #交换第2个和第3个维度`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 拓展/缩减维度           | `在第0个维度，比如3，4变成1，3，4`<br>`x_0 = x.unsqueeze(0)`<br>`在第0个维度，比如1，3，4变成3，4`<br>`如果不指定，则会缩减所有大小为1的维度`<br>`y = x.squeeze(dim=0)`                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 数学运算              | `a = torch.ones((2,3))`  <br>`b = torch.ones((2,3))`<br><br>`print(a + b)  # 加法`<br>`print(a - b)  # 减法`<br>`print(a * b)  # 逐元素乘法`<br>`print(a / b)  # 逐元素除法`<br>`print(a @ b.t())  # 矩阵乘法`                                                                                                                                                                                                                                                                                                                                                               |
| 统计函数              | `tensor.sum()求和，通过tensor.mean()求均值，通过tensor.std()求标准差，通过tensor.min()求最小值等`<br><br>`第0个维度是行，第1个维度是列，mean(dim=0)意味着求行的平均值，mean()意味着求所有元素的平均值`                                                                                                                                                                                                                                                                                                                                                                                                                |
| 索引切片              | `x = torch.tensor([[1, 2, 3], [4, 5, 6]])`  <br>`print(x[0, 1])  # 访问第一行第二个元素`<br>`print(x[:, 1])  # 访问第二列`  <br>`print(x[1, :])  # 访问第二行`  <br>`print(x[:, :2])  # 访问前两列`                                                                                                                                                                                                                                                                                                                                                                                 |
| 广播扩展<br><br>t1+t2 | `例1：`<br>`t1的shape为（3,2,2）`<br>`t2的shape为（1,1,1）`<br>`扩展t2的维度为（3,2,2）`<br><br>`例2：`<br>`t1的shape为（1,2,2）`<br>`t2的shape为（3,2,2）`<br>`扩展t1的维度为（3,2,2）`<br><br>`例3：`<br>`t1的shape为（3,1）`<br>`t2的shape为（1,4）`<br>`扩展t1的维度为（3,4）扩展t2的维度为（3,4）`<br><br>`例4：`<br>`t1的shape为（3,2）`<br>`t2的shape为（1,3）`<br>`因为t1和t2最后一个维度不同，且都不为1，无法进行扩展。所以无法进行广播，无法进行按位计算`                                                                                                                                                                                                         |
| 搬到GPU             | `device = torch.device("cuda" if torch.cuda.is_available() else "cpu")`<br>`size = 10000  # 矩阵大小`<br>`A_cpu = torch.rand(size, size) # 默认在CPU上创建tensor`<br>`A_gpu = A_cpu.to(device) # 将tensor转移到GPU上`                                                                                                                                                                                                                                                                                                                                                     |
| 自动求梯度             | $z=log(3x+4y)^2$<br><br>`x = torch.tensor(1.0, requires_grad=True) #指定需要计算梯度`  <br>`y = torch.tensor(1.0, requires_grad=True) #指定需要计算梯度`<br>`v = 3*x+4*y`  <br>`u = torch.square(v)`  <br>`z = torch.log(u)`<br><br>`z.backward() #反向传播求梯度`<br><br>`print("x grad:", x.grad)`  <br>`print("y grad:", y.grad)`<br><br>`最终打印出的x和y在（1,1）点的梯度和我们手动计算的梯度值是完全一致的`                                                                                                                                                                                                |
